Customer Shopping Behavior Analysis

A complete end-to-end data analytics project focused on understanding customer purchasing behavior, identifying revenue-driving factors, and generating actionable business insights using Python, PostgreSQL, and Power BI.

📌 Project Overview

This project analyzes customer shopping behavior using transactional retail data containing 3,900 customer purchases across multiple product categories.

The objective was to:

Clean and preprocess raw data
Perform exploratory data analysis (EDA)
Execute business-driven SQL analysis
Build an interactive Power BI dashboard
Generate strategic business recommendations

🗄️ PostgreSQL Business Analysis

The cleaned dataset was loaded into PostgreSQL for advanced querying and business analysis.

Business Questions Solved
1. Revenue by Gender

Compared total revenue generated by male vs female customers.

2. High-Spending Discount Users

Identified customers who spent above-average amounts despite using discounts.

3. Top Rated Products

Found products with the highest average customer ratings.

4. Shipping Type Comparison

Compared average purchase amounts across shipping methods.

5. Subscribers vs Non-Subscribers

Analyzed spending patterns based on subscription status.

6. Discount-Dependent Products

Identified products most frequently purchased with discounts.

7. Customer Segmentation

Segmented customers into:

New
Returning
Loyal
8. Top Products per Category

Extracted top-performing products in each category.

9. Repeat Buyers & Subscriptions

Analyzed whether repeat buyers are more likely to subscribe.

10. Revenue by Age Group

Measured revenue contribution across different customer age segments.
